# **App Name**: FelpaChoice

## Core Features:

- Felpa Form: Form for collecting user name, phone number, and choice of sweatshirt (default or with zip).
- Data Submission: Submit collected form data, including name, phone number, and selected sweatshirt type (default/zip) to PocketBase collection pdg_servizio_felpa as a JSON object, following the PocketBase API at https://pocketbase.eulab.cloud/.
- Data List: Display a list of entries from the PocketBase pdg_servizio_felpa collection, showing details such as name, phone, and selected sweatshirt type, fetched using the PocketBase API.
- Data Search: Implement search functionality to find entries in the PocketBase pdg_servizio_felpa collection by name or other relevant fields, utilizing PocketBase API for filtering. This search function uses string matching and basic search functionality.
- Data Filtering: Add filtering mechanism to list by groups inside the pdg_servizio_felpa collection.

## Style Guidelines:

- Primary color: Light blue (#A7D9ED), reminiscent of comfortable clothing, but not cliche.
- Background color: Very light blue (#F0F8FF), the same hue as the primary.
- Accent color: A muted violet (#9486B7) provides a contrast for highlights and actions.
- Body and headline font: 'PT Sans', a sans-serif providing a balance of modern clarity with some warmth.
